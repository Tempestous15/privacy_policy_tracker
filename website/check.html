Check a Site

A lighter version of the extension's scan, without installing anything. Paste a domain to see what we can show from your browser alone.

🔒

Nothing you type here is sent to us. This page makes at most one direct request from your browser to ToS;DR's public API (never through any server of ours) and looks up the domain in a small dataset bundled with this page. See Privacy Policy for exact detail, and note the scope limits below -- this page can't do everything the extension can.

Scope limits, honestly stated: this page can only show a ToS;DR community rating (for sites ToS;DR has reviewed) and Observed data for a small, curated set of sites (the same ~20-site snapshot the extension falls back to). It cannot fetch and locally scan an arbitrary site's actual policy text the way the extension does -- browsers block a plain webpage from fetching another site's pages directly (the extension can, via a permission only an installed extension can request). For a full scan of any site, including live tracker detection, get the extension.
